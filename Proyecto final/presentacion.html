Presentación

Esta web es el proyecto final de la formación Programación Front End de Factoria F5.

A lo largo de este curso presencial de 120h hemos aprendido a construir desde cero páginas web dinámicas, atractivas y responsive, de forma muy práctica usando HTML, CSS y JavaScript. También hemos usado herramientas como Trello, Figma para el diseño UI/UX, GitHub y Visual Studio Code para la programación, y hemos aprendido a gestionar los proyectos con metodologías Agile.
